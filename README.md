# XingBoot-Backend

## 项目背景
方便自己快速开发项目，提升开发效率。抽象出一个自己熟悉、易用的后端框架。

## 项目简介

基于 Spring Boot2 + MySQL + MyBatis-plus + Mybatis-Plus-Join + Swagger2 + Hutool + Mapstruct 等技术后端框架


## 技术选型

#### 后端

- Spring Boot 2.7
- MySQL
- Mybatis-plus
- Mybatis-Plus-Join
- Swagger 接口文档
- Hutool 、Mapstruct  等工具库

## 已有功能模块

#### 用户模块
- 注册登录功能

## 后续规划
- 权限校验
- 日志记录
- 验证码
- Redis、Es
- 前端UI[element-plus-admin]
